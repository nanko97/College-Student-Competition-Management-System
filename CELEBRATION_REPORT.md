```
学生登录 → 我的团队 → 创建团队 → 选择竞赛和赛道 
→ 填写团队信息 → 提交申请 → 等待审核
```

### 2. 人员变更流程
```
学生申请 → 选择团队 → 选择变更类型 → 填写内容和原因 
→ 提交申请 → 教师审核 → 通过/驳回 → 通知结果
```

### 3. 费用配置流程
```
管理员登录 → 费用配置 → 新增配置 → 选择竞赛 
→ 设置费用和日期 → 保存 → 生效
```
