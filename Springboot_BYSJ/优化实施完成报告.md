## 八、常见问题

### Q1: Swagger UI 无法访问？

**A**: 检查以下几点：
1. 项目是否正常启动
2. 访问路径是否正确：http://localhost:8080/springbootrd362/swagger-ui.html
3. 查看控制台是否有报错信息

### Q2: Redis 连接失败？

**A**: 如果未安装 Redis，系统仍可正常运行，只是缓存功能不生效。如需启用：
1. 安装 Redis
2. 启动 Redis 服务
3. 检查 application.yml 中的 Redis 配置

### Q3: 数据库索引如何验证？

**A**: 在 MySQL 中执行：
```sql
SHOW INDEX FROM jingsaibaoming;
SHOW INDEX FROM jingsaixinxi;
```
