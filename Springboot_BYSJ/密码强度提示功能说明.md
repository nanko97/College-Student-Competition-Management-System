# 密码强度提示功能 - 实现说明

**功能名称**: 前端密码强度实时检测与提示  
**实现日期**: 2026 年 3 月 5 日  
**状态**: ✅ 已完成

## 一、功能概述

在用户注册页面，当用户输入密码时，系统会实时检测密码强度并提供可视化反馈，帮助用户设置更安全的密码。

### 核心特性

1. ✅ **实时检测**: 用户输入时即时计算密码强度
2. ✅ **视觉反馈**: 进度条 + 颜色 + 文字提示
3. ✅ **详细要求**: 显示已满足/未满足的字符类型要求
4. ✅ **评分系统**: 0-100 分量化密码强度
5. ✅ **分级提示**: 极弱/弱/中/强 四个等级

## 三、技术实现

### 3.1 密码强度算法

```javascript
calculatePasswordStrength(password) {
  let score = 0;
  
  // 1. 基础分（长度）- 最多 40 分
  score += Math.min(password.length * 4, 40);
  
  // 2. 复杂度分 - 最多 50 分
  if (/[A-Z]/.test(password)) score += 10;  // 大写字母
  if (/[a-z]/.test(password)) score += 10;  // 小写字母
  if (/\d/.test(password)) score += 10;     // 数字
  if (/[^A-Za-z0-9]/.test(password)) score += 20;  // 特殊字符
  
  // 3. 额外加分（长度超过 12）- 10 分
  if (password.length >= 12) {
    score += 10;
  }
  
  return Math.min(score, 100);
}
```

### 3.2 强度等级判定

```javascript
if (score < 30) {
  level = 'weak';       // 极弱
} else if (score < 60) {
  level = 'medium';     // 弱
} else if (score < 80) {
  level = 'strong';     // 中
} else {
  level = 'very-strong'; // 强
}
```

### 3.3 字符类型检测

```javascript
// 实时检测密码包含的字符类型
hasUpperCase: /[A-Z]/.test(password),   // 大写字母
hasLowerCase: /[a-z]/.test(password),   // 小写字母
hasDigit: /\d/.test(password),          // 数字
hasSpecialChar: /[^A-Za-z0-9]/.test(password)  // 特殊字符
```

### 4.2 数据定义

    // ... 其他数据

    showPasswordStrength: false,

### 4.3 计算方法

```javascript
computed: {
  passwordStrengthText() {
    const levelMap = {
      'weak': '极弱',
      'medium': '弱',
      'strong': '中',
      'very-strong': '强'
    };
    return levelMap[this.passwordStrengthLevel] || '极弱';
  }
}
```

### 4.4 事件处理

```javascript
methods: {
  onPasswordInput() {
    const password = this.form.password;
    
    if (!password) {
      this.showPasswordStrength = false;
      return;
    }
    
    this.showPasswordStrength = true;
    
    // 检测字符类型
    this.hasUpperCase = /[A-Z]/.test(password);
    this.hasLowerCase = /[a-z]/.test(password);
    this.hasDigit = /\d/.test(password);
    this.hasSpecialChar = /[^A-Za-z0-9]/.test(password);
    
    // 计算强度分数
    this.passwordStrengthScore = this.calculatePasswordStrength(password);
    
    // 确定强度等级
    if (this.passwordStrengthScore < 30) {
      this.passwordStrengthLevel = 'weak';
    } else if (this.passwordStrengthScore < 60) {
      this.passwordStrengthLevel = 'medium';
    } else if (this.passwordStrengthScore < 80) {
      this.passwordStrengthLevel = 'strong';
    } else {
      this.passwordStrengthLevel = 'very-strong';
    }
  },
  
  getBarClass(index) {
    const activeBars = Math.ceil(this.passwordStrengthScore / 25);
    const levelColors = {
      1: 'bar-weak',
      2: 'bar-medium',
      3: 'bar-strong',
      4: 'bar-very-strong'
    };
    
    if (index < activeBars) {
      return levelColors[activeBars] || 'bar-weak';
    }
    return '';
  }
}
```
