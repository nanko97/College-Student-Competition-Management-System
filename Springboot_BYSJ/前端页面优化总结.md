# 前端页面优化总结

**优化日期**: 2026 年 3 月 6 日  
**优化目标**: 解决页面对齐问题和元素遮挡问题

---

## 一、发现的问题

### 1.1 对齐问题

1. **侧边栏菜单对齐**
   - 侧边栏顶部边距过大（80px），导致与头部导航不对齐
   - 侧边栏宽度不一致（217px vs 200px）

2. **主内容区域对齐**
   - 内边距不统一（0 24px vs 0）
   - 面包屑导航顶部边距导致位置偏移

3. **表单元素对齐**
   - 搜索表单按钮间距不一致
   - 表格按钮组间距不均匀

4. **表格对齐**
   - 表格内容可能超出容器
   - 分页器对齐方式不统一

### 1.2 遮挡问题

1. **侧边栏遮挡**
   - 侧边栏固定定位不当，滚动时位置偏移
   - 侧边栏与主内容区域重叠

2. **下拉菜单遮挡**
   - 下拉菜单 z-index 层级不够，被其他元素遮挡
   - 对话框弹出时被表格覆盖

3. **消息提示遮挡**
   - Message 和 Notification 的 z-index 不够高

4. **表格表头遮挡**
   - 表格滚动时表头未固定，导致表头被内容遮挡

## 二、已实施的优化

### 2.1 新增全局样式修复文件

**文件**: `src/assets/css/common-fix.css`

**主要修复内容**:

```css
/* 基础布局修复 */
html, body {
  overflow-x: hidden; /* 防止横向滚动条 */
}

/* Element UI 容器修复 */
.el-container {
  height: 100%;
  width: 100%;
}

.el-header {
  z-index: 1000; /* 确保头部在最上层 */
}

/* 侧边栏修复 */
.el-aside {
  overflow-y: auto;
  overflow-x: hidden;
  z-index: 999;
}

/* 主内容区域修复 */
.el-main {
  min-height: calc(100vh - 60px);
  overflow: auto;
}

/* 面包屑导航修复 */
.bread-crumbs {
  position: sticky;
  top: 0;
  z-index: 100;
}

/* 表格容器修复 */
.table-content {
  overflow-x: auto; /* 允许横向滚动 */
}

/* 对话框修复 */
.el-dialog {
  max-height: 90vh;
}

.el-dialog__body {
  max-height: 70vh;
  overflow-y: auto;
}

/* 下拉菜单层级修复 */
.el-dropdown-menu,
.el-select-dropdown {
  z-index: 2000 !important;
}

/* 消息提示层级修复 */
.el-message {
  z-index: 3000 !important;
}

.el-notification {
  z-index: 3000 !important;
}
```

### 2.2 修改 IndexAside.vue

**修改前**:
```scss
.index-aside {
  margin-top: 80px;  // ❌ 过大的顶部边距
  overflow: hidden;  // ❌ 隐藏溢出，无法滚动
  
  .index-aside-inner {
    width: 217px;    // ❌ 宽度不一致
    height: 100%;
    overflow-y: scroll;
  }
}
```

**修改后**:
```scss
.index-aside {
  margin-top: 0 !important;           // ✅ 移除顶部边距
  height: calc(100vh - 60px) !important; // ✅ 减去头部高度
  position: fixed !important;         // ✅ 固定定位
  left: 0;
  top: 60px;                          // ✅ 从头部下方开始
  z-index: 999;
  
  .index-aside-inner {
    width: 200px;                     // ✅ 统一宽度
    height: 100%;
    overflow-y: auto;                 // ✅ 自动滚动
    overflow-x: hidden;
    
    // ✅ 美化滚动条
    &::-webkit-scrollbar {
      width: 6px;
    }
  }
}
```

### 2.3 修改 IndexMain.vue

**修改前**:
```scss
.el-main {
  background-color: #F6F8FA;
  padding: 0 24px;        // ❌ 内边距不统一
}

.bread-crumbs {
  margin-top: 10px;       // ❌ 顶部边距导致偏移
}
```

**修改后**:
```scss
.el-main {
  background-color: #F6F8FA;
  padding: 0 !important;     // ✅ 移除默认内边距
  margin-left: 200px;        // ✅ 侧边栏宽度
  min-height: 100vh;         // ✅ 最小高度
}

.bread-crumbs {
  margin-top: 0;             // ✅ 移除顶部边距
  position: sticky;          // ✅ 固定定位
  top: 0;
  z-index: 100;
  background: #fff;
}
```

### 2.4 在 main.js 中引入全局样式

**修改**:
```javascript
// 新增引入全局样式修复文件
import '@/assets/css/common-fix.css'
```

## 七、常见问题

### Q1: 侧边栏为什么不显示？

**A**: 检查 `IndexAside.vue` 的 `top` 值是否与头部高度一致（默认 60px）。

### Q2: 表格内容被截断？

**A**: 确保 `.table-content` 设置了 `overflow-x: auto`。

### Q3: 下拉菜单仍然被遮挡？

**A**: 检查是否有其他元素的 z-index 超过 2000，必要时调整。

### Q4: 移动端布局异常？

**A**: 清除浏览器缓存，确保 CSS 文件已更新。
