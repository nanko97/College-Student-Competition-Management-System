**功能说明**:

- 4 级强度评价：极弱/弱/中/强

**技术实现**:
```javascript
// 密码强度算法
calculatePasswordStrength(password) {
  let score = 0;
  score += Math.min(password.length * 4, 40);  // 长度分
  if (/[A-Z]/.test(password)) score += 10;     // 大写字母
  if (/[a-z]/.test(password)) score += 10;     // 小写字母
  if (/\d/.test(password)) score += 10;        // 数字
  if (/[^A-Za-z0-9]/.test(password)) score += 20; // 特殊字符
  return Math.min(score, 100);
}
```

#### ✅ 2. XSS 跨站脚本攻击防护

**文件**: 
- `src/main/java/com/filter/XssHttpServletRequestWrapper.java`
- `src/main/java/com/filter/XssFilter.java`

**功能说明**:
- 拦截所有 HTTP 请求
- 自动过滤 HTML 标签和 JavaScript 代码
- 防止恶意脚本注入
- 保护用户数据安全

**过滤规则**:
```java
// 1. 转义 HTML 特殊字符
value = value.replaceAll("<", "&lt;")
             .replaceAll(">", "&gt;");

// 2. 转义引号和括号
value = value.replaceAll("\\(", "&#40;")
             .replaceAll("'", "&#39;");

// 3. 过滤 javascript:协议
value = value.replaceAll("(?i)javascript:", "");

// 4. 过滤事件处理器
value = value.replaceAll("(?i)\\bon\\w+\\s*=", "");
```

**防护场景**:
| 攻击类型 | 示例 | 防护效果 |
|----------|------|----------|
| HTML 注入 | `<script>alert('xss')</script>` | ✅ 已过滤 |
| 事件注入 | `<img onerror="alert(1)">` | ✅ 已过滤 |
| 协议注入 | `javascript:alert(1)` | ✅ 已过滤 |

**毕业答辩价值**:
- 展示网络安全意识
- 符合 OWASP Top 10 防护要求
- 提升系统安全性评分
