## 📝 方法三：通过系统界面添加数据

### 1. 管理员登录

- **地址**: http://localhost:8081/#/login
- **账号**: admin / admin123

### 2. 添加学生

1. 进入 **学生管理** 页面
2. 点击 **添加** 按钮
3. 填写信息：
   - 学号：2022051 ~ 2022100
   - 姓名：随机中文名字
   - 性别：男/女
   - 学院：计算机学院、软件学院、信息学院等
   - 班级：专业 + 年级 + 班号
   - 手机号：138xxxxxxx
   - 邮箱：学号@school.edu.cn

### 3. 添加教师

1. 进入 **教师管理** 页面
2. 点击 **添加** 按钮
3. 填写信息：
   - 工号：T2022021 ~ T2022050
   - 姓名：中文名字 + 老师
   - 性别：男/女
   - 学院：同学生学院
   - 职称：教授、副教授、讲师、助教

### 4. 添加竞赛

1. 进入 **竞赛信息管理** 页面
2. 点击 **添加** 按钮
3. 填写信息：
   - 竞赛名称：如"全国大学生 XX 竞赛"
   - 竞赛类型：学科竞赛/创新创业/科技创新/艺术设计/体育竞技
   - 竞赛地点：校内教室、实验室、报告厅等
   - 竞赛规则：详细描述比赛规则
   - 竞赛奖励：一二三等奖设置
   - 竞赛时间：2026 年各月份
   - 模式：个人赛/团队赛
   - 封面图片：上传生成的图片

### 5. 添加报名

1. 进入 **竞赛报名管理** 页面
2. 点击 **添加** 按钮
3. 选择学生和竞赛
4. 填写参赛信息：
   - 参赛类型：个人/团队
   - 参赛人员：1-5 人
   - 参赛作品：作品名称
   - 参赛宣言：激励性语句

### 6. 添加评分

1. 进入 **作品打分管理** 页面
2. 点击 **添加** 按钮
3. 选择学生和竞赛
4. 填写评分信息：
   - 作品评分：60-100 分
   - 评价内容：针对性评语
   - 评价时间：竞赛时间之后

## 🔧 常见问题

### Q1: MySQL 命令无法执行？

**解决方案 1**: 使用完整路径
```bash
"C:\Program Files\MySQL\MySQL Server 8.0\bin\mysql.exe" -u root -p springbootrd362 < "src/main/resources/db/migration/V3__add_test_data.sql"
```

**解决方案 2**: 使用 Navicat、DBeaver 等数据库工具直接打开 SQL 文件并执行

**解决方案 3**: 在 phpMyAdmin 中导入 SQL 文件

### Q2: 图片无法显示？

**原因**: 图片路径不正确或文件不存在

**解决方案**:
1. 确保 `upload` 目录在项目根目录
2. 检查图片文件名是否与数据库记录一致
3. 重启后端服务

### Q3: 如何清除测试数据？

**方案**: 执行以下 SQL 清空数据（谨慎操作！）
```sql
TRUNCATE TABLE xuesheng;
TRUNCATE TABLE jiaoshi;
TRUNCATE TABLE jingsaixinxi;
TRUNCATE TABLE jingsaibaoming;
TRUNCATE TABLE zuopindafen;
```
