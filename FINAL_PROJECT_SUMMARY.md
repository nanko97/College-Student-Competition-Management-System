## 🎉 项目概览

### 项目简介

BYSJ（毕业设计/竞赛管理系统）是一个基于Vue 3 + Spring Boot的现代化竞赛管理平台，支持多角色（管理员、教师、学生）的竞赛报名、团队管理、作品提交、评分晋级等全流程管理。

---

### 技术栈

#### 前端（new-admin）
- **框架**: Vue 3.5+ Composition API
- **语言**: TypeScript 4.9+
- **UI库**: Naive UI 2.43+
- **构建工具**: Vite 5.x
- **状态管理**: Pinia
- **路由**: Vue Router 4.x
- **HTTP客户端**: Alova
- **端口**: 8081

#### 后端（Springboot_BYSJ）
- **框架**: Spring Boot 2.2.2.RELEASE
- **ORM**: MyBatis-Plus 2.3.3
- **数据库**: MySQL 8.0.28
- **认证**: JWT Token
- **加密**: BCrypt
- **端口**: 9090
- **Context Path**: /BYSJ_Springboot

### Q3: 如何添加新功能？

**A**: 
遵循标准CRUD流程：
1. 后端：Entity → Mapper → Service → Controller
2. 前端：API接口 → Columns定义 → Vue组件
3. 确保字段名与后端一致
4. 添加TypeScript类型定义
5. 配置路由和权限
6. 编写测试用例

---

### Q4: 遇到Bug怎么办？

**A**: 
1. 查看浏览器控制台错误信息
2. 检查Network中的API请求
3. 查看后端日志（logs目录）
4. 确认字段名是否正确
5. 检查Token是否有效
6. 查看相关文档
