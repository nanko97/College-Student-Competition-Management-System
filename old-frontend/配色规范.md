# 前端统一配色规范

## 📋 配色方案概览

本项目采用 **深色赛博朋克主题**，基于 BYSJ_02 设计风格。

### 核心颜色

```scss
// 主背景色
$bg-primary: #0a0e27;     // 深蓝黑（侧边栏、Header）
$bg-secondary: #1a202c;   // 次级背景
$bg-light: #f6f8fa;       // 浅灰背景（主内容区）
$bg-white: #ffffff;       // 白色背景（卡片、内容区）

// 品牌色
$brand-primary: #667eea;    // 紫色
$brand-secondary: #764ba2;  // 紫罗兰
$brand-gradient: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

// 功能色
$success-color: #51cf66;  // 薄荷绿
$warning-color: #fcc419;  // 明黄色
$danger-color: #ff6b6b;   // 珊瑚红
$info-color: #339af0;     // 蓝色
```

## 🎯 使用方式

### 1. 在 Vue 组件中使用变量

```vue
<style lang="scss" scoped>
.my-card {
  background: $bg-white;
  border-radius: $radius-lg;
  box-shadow: $shadow-md;
  padding: $spacing-xl;
  
  .title {
    color: $text-primary;
    font-size: $font-size-xl;
    @include gradient-text;  // 渐变文字
  }
}
</style>
```

### 2. 使用混合器

#### 卡片样式
```scss
.my-card {
  @include card;
}
```

#### 按钮样式
```scss
.my-button {
  @include button-primary;
}
```

#### 输入框样式
```scss
.my-input {
  @include input;
}
```

#### 标签样式
```scss
.tag-success {
  @include tag('success');
}
.tag-warning {
  @include tag('warning');
}
.tag-danger {
  @include tag('danger');
}
.tag-info {
  @include tag('info');
}
.tag-default {
  @include tag('default');
}
```

### 3. 统一圆角规范

```scss
$radius-sm: 4px;   // 小圆角
$radius-md: 8px;   // 中等圆角（按钮、输入框）
$radius-lg: 12px;  // 大圆角（卡片）
$radius-xl: 16px;  // 超大圆角（对话框）
```

### 4. 统一间距规范

```scss
$spacing-xs: 4px;   // 极小间距
$spacing-sm: 8px;   // 小间距
$spacing-md: 16px;  // 中等间距
$spacing-lg: 24px;  // 大间距
$spacing-xl: 32px;  // 超大间距
$spacing-xxl: 48px; // 极大间距
```

### 5. 统一字体规范

```scss
$font-size-xs: 12px;   // 极小字体（标签）
$font-size-sm: 13px;   // 小字体
$font-size-md: 14px;   // 标准字体
$font-size-lg: 16px;   // 大字体
$font-size-xl: 18px;   // 超大字体（标题）
$font-size-xxl: 24px;  // 极大字体
$font-size-xxxl: 32px; // 标题字体

$font-weight-normal: 400;     // 正常
$font-weight-medium: 500;     // 中等
$font-weight-semibold: 600;   // 半粗
$font-weight-bold: 700;       // 粗体
```

## 🎨 示例：新建页面

```vue
<template>
  <div class="my-page">
    <!-- 页面标题 -->
    <div class="page-header">
      <h2 class="page-title">我的页面</h2>
    </div>
    
    <!-- 内容卡片 -->
    <div class="content-card">
      <div class="card-body">
        <p>这里是内容</p>
        <el-button type="primary">操作按钮</el-button>
      </div>
    </div>
  </div>
</template>

<style lang="scss" scoped>
.my-page {
  padding: $spacing-xl;
  
  .page-header {
    margin-bottom: $spacing-lg;
    
    .page-title {
      font-size: $font-size-xxl;
      font-weight: $font-weight-bold;
      color: $text-primary;
      @include gradient-text;  // 渐变文字效果
    }
  }
  
  .content-card {
    @include card;  // 使用统一卡片样式
    
    .card-body {
      display: flex;
      align-items: center;
      justify-content: space-between;
      gap: $spacing-md;
    }
  }
}
</style>
```

## ✅ 检查清单

新建页面时，请确保：

- [ ] 使用 `$bg-white` 作为内容区背景
- [ ] 使用 `$brand-gradient` 作为按钮渐变背景
- [ ] 使用 `$radius-lg` 作为卡片圆角
- [ ] 使用 `$shadow-md` 作为卡片阴影
- [ ] 使用 `$spacing-xl` 作为页面内边距
- [ ] 使用 `@include card` 混合器简化代码
- [ ] 使用 `$text-primary` 作为主文字颜色
- [ ] 所有按钮使用统一的圆角和阴影
