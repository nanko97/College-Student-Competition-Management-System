📋 检查步骤

请按 F12 打开浏览器开发者工具，然后：

  1. 访问 http://localhost:8081/
  2. 登录系统（教师账号：T2022001 / 123456）
  3. 进入"竞赛信息管理"页面
  4. 按 F12 打开 Console（控制台）
  5. 查看是否有以下日志：
    • "开始加载竞赛列表..."
    • "请求参数：..."
    • "竞赛列表响应数据：..."
    • "最终 dataList：..."
  6. 如果没有看到日志，说明页面代码有问题
  7. 如果看到日志但 dataList 为空数组，请把日志内容复制给我

💡 常见问题排查

现象 可能原因 解决方案
页面显示"共 X 条"但没有数据 浏览器缓存旧代码 Ctrl + Shift + Delete 清除缓存，然后 Ctrl + F5 强制刷新
控制台有 "$storage is not defined" 错误 代码未正确编译 重新编译前端并清除缓存
控制台有 "401 Unauthorized" 错误 Token 失效或未登录 重新登录
控制台有 "Network Error" 后端服务未启动 检查后端是否在 8080 端口运行
表格完全不可见 CSS 样式问题或被隐藏 检查元素查看表格是否存在
