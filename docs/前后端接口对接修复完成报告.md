5. **确认数据能正常加载和显示**

### 5.2 检查浏览器控制台

1. 按 `F12` 打开开发者工具
2. 切换到 **Network** 标签
3. 刷新列表页面
4. 查看API请求：
   - 状态码应为 `200` (成功)
   - 响应数据应包含 `page.list` 数组
   - 数组中应有数据对象

### 5.3 常见问题排查

**问题1**: 列表显示为空

**解决步骤**:
1. 检查是否已登录
2. 检查浏览器控制台是否有错误
3. 检查 Network 面板中API请求的状态码
4. 确认当前用户的角色和权限

**问题2**: 接口返回401

**原因**: 未登录或token过期

**解决**: 重新登录系统

**问题3**: 接口返回404

**原因**: 接口路径错误或后端服务未启动

**解决**: 
1. 确认后端服务正在运行 (`http://localhost:9090`)
2. 检查API路径是否正确
3. 查看后端日志是否有错误
