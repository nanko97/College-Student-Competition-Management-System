## 🎯 核心模块说明

### 1. 后端模块 (Spring Boot)

#### Controller层 - 控制器
负责接收HTTP请求，调用Service层处理业务逻辑

主要Controller：
- `UserController` - 用户管理
- `XueshengController` - 学生管理
- `JiaoshiController` - 教师管理
- `JingsaixinxiController` - 竞赛信息管理
- `JingsaibaomingController` - 竞赛报名管理
- `JingsaiJiaofeiController` - 缴费管理
- `JingsaiTuanduiController` - 团队管理
- `JingsaiRenyuanBianguengController` - 人员变更管理
- `JingsaiJinjiController` - 晋级管理
- `JingsaiSaidaoController` - 赛道管理
- `ZuopindafenController` - 作品打分管理

#### Service层 - 业务逻辑
实现核心业务逻辑，事务控制

#### Dao层 - 数据访问
使用MyBatis-Plus进行数据库操作

#### Entity层 - 实体类
对应数据库表的Java对象

#### Config层 - 配置类
- `WebMvcConfig` - Web MVC配置（静态资源映射）
- `RedisConfig` - Redis缓存配置
- `MybatisPlusConfig` - MyBatis-Plus配置
- `InterceptorConfig` - 拦截器配置
- `GlobalExceptionHandler` - 全局异常处理

### 2. 前端模块 (Vue.js + Element UI)

位置：`Springboot_BYSJ/src/main/resources/admin/admin/`

主要页面：
- `views/modules/xuesheng/` - 学生端页面
  - `my-jiaofei.vue` - 我的缴费
  - `pingzheng-viewer.vue` - 凭证查看器
- `views/modules/jingsai-jiaofei/` - 缴费管理页面
  - `list.vue` - 缴费列表
  - `shenhe-list.vue` - 审核列表
- `views/modules/jingsaixinxi/` - 竞赛信息页面
- `views/modules/jingsaibaoming/` - 报名管理页面

### 3. 数据库模块

数据库名称：`bysj_springboot`

主要数据表：
- `users` - 系统用户表
- `xuesheng` - 学生信息表
- `jiaoshi` - 教师信息表
- `jingsaixinxi` - 竞赛信息表
- `jingsaibaoming` - 竞赛报名表
- `jingsai_jiaofei_jilu` - 缴费记录表
- `jingsai_tuandui` - 团队表
- `jingsai_renyuan_biangueng` - 人员变更记录表
- `jingsai_jinji_guanxi` - 晋级关系表

## 🚀 快速开始

### 环境要求
- JDK 1.8+
- Maven 3.6+
- Node.js 14+
- MySQL 8.0+
- Redis（可选）

### 启动步骤

#### 1. 数据库准备
```bash
# 创建数据库
mysql -u root -p
CREATE DATABASE bysj_springboot CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

# 导入数据（如果有SQL文件）
mysql -u root -p bysj_springboot < database.sql
```

#### 2. 配置修改
编辑 `Springboot_BYSJ/src/main/resources/application.yml`
```yaml
spring:
  datasource:
    username: root      # 修改为你的数据库用户名
    password: 123123    # 修改为你的数据库密码
```

#### 3. 启动后端
```bash
cd Springboot_BYSJ
.\mvnw spring-boot:run
```

访问：http://localhost:8080/BYSJ_Springboot

#### 4. 启动前端（开发模式）
```bash
cd Springboot_BYSJ/src/main/resources/admin/admin
npm install
npm run serve
```

## ⚠️ 注意事项

### 1. 文件上传
- 上传文件保存在：`src/main/resources/static/upload/`
- 确保该目录有写入权限
- 支持的文件类型：jpg, png, jpeg, gif, pdf, doc, docx
- 单个文件最大：10MB

### 2. 日志文件
- 日志目录：`logs/`
- 会自动创建，无需手动建立
- 定期清理旧日志以节省空间

### 3. 端口配置
- 默认端口：8080
- 如需修改，编辑 `application.yml` 中的 `server.port`

### 4. 跨域配置
- 已配置CORS支持
- 前端可以直接访问后端API

## 🐛 常见问题

### Q1: 启动时提示端口被占用？
**A**: 修改 `application.yml` 中的 `server.port` 为其他端口，如8081

### Q2: 数据库连接失败？
**A**: 
1. 检查MySQL服务是否启动
2. 确认数据库名称、用户名、密码正确
3. 检查 `application.yml` 中的配置

### Q3: 图片上传后无法显示？
**A**: 
1. 检查 `WebMvcConfig` 中的静态资源映射
2. 确认文件已保存到 `static/upload/` 目录
3. 浏览器访问：http://localhost:8080/BYSJ_Springboot/upload/文件名

### Q4: 前端页面一直加载？
**A**: 
1. 检查后端服务是否正常运行
2. 打开浏览器控制台查看错误信息
3. 确认API地址配置正确
