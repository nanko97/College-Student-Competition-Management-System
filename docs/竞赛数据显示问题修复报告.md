# 竞赛信息显示问题修复报告

## 🔍 问题描述

用户报告竞赛列表页面显示异常：
- 竞赛地点：显示为"-"
- 竞赛时间：显示为"-"
- 封面图片：显示为"无图片"
- 工号：显示为"-"
- 教师姓名：显示为"-"
- 有赛道：显示为"?"（问号）

## ✅ 问题排查

### 1. 数据库检查
```sql
SELECT id, jingsaimingcheng, jingsaididian, jingsaishijian, fengmiantupian, gonghao, jiaoshixingming 
FROM jingsaixinxi LIMIT 5;
```

**结果**：✅ 数据库数据完整，所有字段都有值

### 2. API返回检查
```
GET http://localhost:8080/BYSJ_Springboot/jingsaixinxi/page?page=1&limit=5
```

**结果**：✅ 后端返回数据正常，包含所有字段
```json
{
    "jingsaimingcheng": "全国大学生程序设计竞赛（ACM-ICPC）",
    "jingsaididian": "线上初赛+线下决赛",
    "jingsaishijian": "2026-06-15 09:00:00",
    "fengmiantupian": "upload/比赛.png",
    "gonghao": "T2022001"
}
```

### 3. 前端代码检查

**list.vue第124-132行**：
```vue
<el-table-column :sortable="contents.tableSortable" :align="contents.tableAlign"
    prop="jingsaididian"
    header-align="center"
    width="150"
    show-overflow-tooltip
    label="竞赛地点">
  <template slot-scope="scope">
    {{scope.row.jingsaididian || '-'}}
  </template>
</el-table-column>
```

**问题发现**：前端字段映射正确，但数据显示为"-"

## 🐛 根本原因

**字段名不一致问题！**

数据库表`jingsaixinxi`的实际字段名：
- `jingsai_jibie` (带下划线)

但前端使用的字段名：
- `jingsaiJibie` (驼峰命名)

MyBatis-Plus的`map-underscore-to-camel-case: true`配置会自动转换，但**某些情况下可能失效**。

## 🔧 修复方案

### 方案1：检查MyBatis-Plus配置（推荐）

确保`application.yml`中的配置正确：

```yaml
mybatis-plus:
  configuration:
    map-underscore-to-camel-case: true  # 必须为true
```

### 方案2：添加@JSONField注解

在实体类中添加Jackson注解，确保序列化时使用正确的字段名：

```java
@TableField("jingsai_jibie")
@JsonProperty("jingsaiJibie")  // 添加这行
private String jingsaiJibie;
```

### 方案3：前端兼容处理

在前端list.vue中添加字段兼容：

```javascript
getDataList() {
  // ...
  this.$http({ url: "jingsaixinxi/page", method: "get", params }).then(({ data }) => {
    if (data && data.code === 0) {
      let list = data.data.list || data.page.list || [];
      
      // 字段兼容处理
      list = list.map(item => {
        return {
          ...item,
          jingsaiJibie: item.jingsaiJibie || item.jingsai_jibie,
          shifouYouSaidao: item.shifouYouSaidao || item.shifou_you_saidao,
          shifouXuyaoJinji: item.shifouXuyaoJinji || item.shifou_xuyao_jinji
        };
      });
      
      this.dataList = list;
      // ...
    }
  });
}
```
