### 2. 操作确认对话框系统

#### 💬 确认对话框Hook (`src/hooks/web/useConfirm.ts`) - 355行

创建了完整的确认对话框和Toast通知系统：

##### 确认对话框函数

| 函数 | 说明 | 使用场景 |
|------|------|---------|
| `showConfirm()` | 通用确认对话框 | 任意需要确认的操作 |
| `showDeleteConfirm()` | 删除确认 | 单条数据删除 |
| `showBatchDeleteConfirm()` | 批量删除确认 | 多条数据删除 |
| `showSubmitConfirm()` | 提交确认 | 表单提交 |
| `showLogoutConfirm()` | 登出确认 | 用户登出 |
| `showResetConfirm()` | 重置确认 | 表单重置 |

##### Toast通知函数

| 函数 | 说明 | 显示时长 |
|------|------|---------|
| `showSuccessToast()` | 成功提示 | 2秒 |
| `showErrorToast()` | 错误提示 | 3秒 |
| `showWarningToast()` | 警告提示 | 3秒 |
| `showInfoToast()` | 信息提示 | 2秒 |
| `showLoadingToast()` | 加载提示 | 手动关闭 |
| `hideLoadingToast()` | 隐藏加载提示 | - |

##### 组合操作函数

| 函数 | 说明 |
|------|------|
| `executeWithConfirm()` | 确认后执行操作 |
| `executeWithLoading()` | 带加载状态执行操作 |
| `executeWithConfirmAndLoading()` | 确认+加载状态执行操作 |

**使用示例**：

```vue
<script setup lang="ts">
import { 
  showDeleteConfirm, 
  showSuccessToast,
  executeWithConfirm 
} from '@/hooks/web/useConfirm'

// 简单删除确认
const handleDelete = async (id: number) => {
  const confirmed = await showDeleteConfirm('竞赛名称')
  if (confirmed) {
    await deleteCompetition([id])
    showSuccessToast('删除成功')
  }
}

// 组合操作（确认+执行+提示）
const handleSubmit = async () => {
  await executeWithConfirm(
    {
      title: '确认提交',
      content: '确定要提交报名表吗？',
      positiveText: '提交',
    },
    '提交成功',
    async () => {
      await submitRegistration(formData)
    }
  )
}
</script>
```

**收益**：
- ✅ 统一的操作确认流程
- ✅ 防止误操作
- ✅ 友好的用户反馈
- ✅ 简化代码逻辑

### 3. 表单交互优化系统

#### 📝 表单交互Hook (`src/hooks/web/useFormInteraction.ts`) - 393行

创建了强大的表单交互增强功能：

##### 自动保存功能 (`useAutoSave`)

**特性**：
- ⏰ 定时自动保存（默认30秒）
- 💾 本地存储草稿
- 🔄 自动恢复草稿
- ⏱️ 24小时过期机制
- 🔧 可配置的保存间隔
- 🎯 保存前后回调

**使用示例**：
```vue
<script setup lang="ts">
import { useAutoSave } from '@/hooks/web/useFormInteraction'

const formData = ref({
  name: '',
  description: '',
})

const {
  hasDraft,
  lastSaveTime,
  startAutoSave,
  manualSave,
  clearDraft,
} = useAutoSave(formData, {
  interval: 30000, // 30秒
  storageKeyPrefix: 'competition_form_',
})

// 启动自动保存
onMounted(() => {
  startAutoSave('new_competition')
})

// 手动保存
const handleManualSave = () => {
  manualSave('new_competition')
}

// 清除草稿
const handleClearDraft = () => {
  clearDraft('new_competition')
}
</script>

<template>
  <div>
    <NForm v-model="formData">
      <!-- 表单字段 -->
    </NForm>
    
    <div v-if="hasDraft" class="draft-info">
      <span>草稿已保存: {{ formatDateTime(lastSaveTime) }}</span>
      <NButton @click="handleClearDraft">清除草稿</NButton>
    </div>
  </div>
</template>
```

```vue

import { showBatchDeleteConfirm, showSuccessToast } from '@/hooks/web'

const selectedIds = ref<number[]>([])

const handleBatchDelete = async () => {
  if (selectedIds.value.length === 0) {
    showWarningToast('请选择要删除的数据')
    return
  }
  
  const confirmed = await showBatchDeleteConfirm(selectedIds.value.length)
  if (confirmed) {
    await deleteCompetitions(selectedIds.value)
    showSuccessToast('删除成功')
    selectedIds.value = []
  }
}
</script>
```
