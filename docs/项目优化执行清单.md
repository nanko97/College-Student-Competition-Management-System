## 🔧 优化模板

### Controller方法优化模板

```java
/**
 * 【功能说明】
 * 
 * 【重要】特殊说明
 * 
 * 业务逻辑：
 * 1. 参数校验
 * 2. 业务验证
 * 3. 执行操作
 * 4. 返回结果
 * 
 * @param xxx 参数说明
 * @return R 统一返回结果
 */
@OperationLog("操作描述")
@RequestMapping("/xxx")
@Transactional(rollbackFor = Exception.class)
public R xxx(@RequestBody XxxEntity entity, HttpServletRequest request) {
    // 定义常量
    final String STATUS_XXX = "状态值";
    
    try {
        log.info("开始xxx操作，参数: {}", entity);
        
        // 1. 参数校验
        Assert.notNull(entity.getId(), "ID不能为空");
        
        // 2. 业务验证
        // ...
        
        // 3. 执行操作
        // ...
        
        log.info("✓ xxx操作成功，ID: {}", entity.getId());
        return R.ok("操作成功");
        
    } catch (IllegalArgumentException e) {
        log.error("✗ xxx操作参数错误: {}", e.getMessage());
        return R.error(e.getMessage());
    } catch (RuntimeException e) {
        log.error("✗ xxx操业务异常", e);
        throw e; // 触发事务回滚
    } catch (Exception e) {
        log.error("✗ xxx操作系统异常", e);
        throw new RuntimeException("操作失败：" + e.getMessage(), e);
    }
}
```
