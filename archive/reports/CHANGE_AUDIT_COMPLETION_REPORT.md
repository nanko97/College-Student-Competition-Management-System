### 2. 教师/管理员审核

#### 审核流程
```
查看待审核列表
    ↓
点击"审核"按钮
    ↓
弹出审核对话框
    ↓
查看变更详情
    ↓
选择审核结果（通过/驳回）
    ↓
填写审核意见（驳回时必填）
    ↓
提交审核
    ↓
后端处理
    ↓
更新列表
```

#### 审核表单验证
```typescript
const auditFormRules = {
  shenheJieguo: [{ required: true }],
  shenheYijian: [
    {
      validator: (rule, value) => {
        // 驳回时必须填写审核意见
        if (auditForm.shenheJieguo === '已驳回' && !value) {
          return new Error('驳回时必须填写审核意见')
        }
        return true
      }
    }
  ]
}
```

## 🚀 测试指南

### 测试步骤

#### 1. 学生申请变更

1. **登录学生账号**
   - 访问 http://localhost:8081
   - 使用学生账号登录

2. **访问人员变更页面**
   - 点击左侧菜单"团队管理" → "人员变更"
   - 应该看到"申请变更"按钮

3. **点击申请变更**
   - 跳转到申请页面
   - 选择团队
   - 选择变更类型
   - 填写变更内容和原因
   - 提交申请

4. **验证结果**
   - 显示"变更申请提交成功，等待审核"
   - 返回变更列表
   - 可以看到新提交的申请，状态为"审核中"

---

#### 2. 教师审核变更

1. **登录教师账号**
   - 使用教师账号登录

2. **访问人员变更页面**
   - 点击左侧菜单"团队管理" → "人员变更"
   - 应该看到"审核变更"按钮

3. **点击审核变更**
   - 跳转到审核页面
   - 看到待审核的变更申请列表

4. **审核操作**
   - 找到状态为"审核中"的申请
   - 点击"审核"按钮
   - 弹出审核对话框
   - 查看变更详情
   - 选择审核结果（通过/驳回）
   - 如果驳回，填写审核意见
   - 提交审核

5. **验证结果**
   - 显示"审核成功"
   - 列表中该申请的状态更新
   - 已通过/已驳回的申请不再显示审核按钮

---

### 预期问题及解决方案

#### 问题1: 看不到审核按钮

**原因**: 当前用户不是教师或管理员  
**解决**: 
- 确认登录的是教师或管理员账号
- 检查localStorage中的userInfo.role字段

---

#### 问题2: 看不到待审核的申请

**原因**: 没有待审核的申请或权限不足  
**解决**: 
- 确认有学生提交了变更申请
- 教师只能看到自己竞赛的申请
- 管理员可以看到所有申请

---

#### 问题3: 审核失败

**原因**: 表单验证失败或后端接口错误  
**解决**: 
- 检查浏览器控制台错误信息
- 驳回时必须填写审核意见
- 检查Network中的API请求
