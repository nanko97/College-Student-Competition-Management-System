## 🎯 快速启动命令汇总

### 一键启动脚本（可选）
创建 `start-all.bat`：
```batch
@echo off
echo ========================================
echo 启动竞赛管理系统
echo ========================================

echo [1/3] 检查MySQL服务...
net start | findstr MySQL
if errorlevel 1 (
    echo MySQL服务未启动，请先启动MySQL
    pause
    exit
)

echo [2/3] 启动后端服务...
start "后端服务" cmd /k "cd /d D:\BYSJ\BYSJ\Springboot_BYSJ && mvn spring-boot:run"

echo [3/3] 启动前端服务...
start "前端服务" cmd /k "cd /d D:\BYSJ\BYSJ\old-frontend && npm run serve"

echo ========================================
echo 启动完成！
echo 前端地址：http://localhost:8080
echo 后端地址：http://localhost:9090
echo ========================================
pause
```
