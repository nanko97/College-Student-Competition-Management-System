# 统计数据展板字段映射修复清单

## 问题描述
多个页面的统计数据展板显示为0，原因是：
1. 前端期望的字段名与后端返回的字段名不匹配
2. 部分后端Controller缺少statistics接口

## 修复原则
1. 为缺少的Controller添加statistics接口
2. 前端`getStatistics()`方法中添加字段映射逻辑
3. 保持前端模板中的字段名不变
4. 在后端返回数据和前端显示之间做适配

---

## 📝 注意事项

1. **教师权限**：教师的statistics接口需要限制只能统计自己创建的竞赛相关数据
2. **管理员权限**：管理员可以统计全局数据
3. **字段命名**：前后端字段名尽量保持一致，减少映射复杂度
4. **空值处理**：使用 `|| 0` 处理空值，确保显示为0而不是undefined
