# 统计卡片样式统一实施计划

## 📋 概述

将所有页面的旧版统计卡片（el-card + 内联样式）替换为新版统一样式（statistics-wrapper + 渐变卡片）

---

## 🎯 修改原则

### 旧版样式（需要删除）
```html
<el-row :gutter="20" class="stats-row">
  <el-col :span="8">
    <el-card shadow="hover" class="stat-card">
      <div class="stat-content">
        <div class="stat-icon" style="background: linear-gradient(...)">
          <i class="el-icon-xxx"></i>
        </div>
        <div class="stat-info">
          <div class="stat-label">标签</div>
          <div class="stat-value">{{ value }}</div>
        </div>
      </div>
    </el-card>
  </el-col>
</el-row>
```

### 新版样式（需要替换为）
```html
<div class="statistics-wrapper">
  <el-row :gutter="20">
    <el-col :span="6">
      <div class="stat-card stat-purple">
        <div class="stat-icon"><i class="el-icon-xxx"></i></div>
        <div class="stat-content">
          <div class="stat-value">{{ value }}</div>
          <div class="stat-label">标签</div>
        </div>
      </div>
    </el-col>
  </el-row>
</div>
```

### 样式引入（在 `<style>` 中添加）
```scss
@import '@/assets/css/statistics-cards.scss';
```

## 📝 配色方案选择指南

| 场景 | 推荐配色类 | 渐变色 |
|------|-----------|--------|
| 总数、总计、全部 | `stat-purple` | 紫色 `#667eea → #764ba2` |
| 待审核、待处理、等待 | `stat-pink` | 粉红 `#f093fb → #f5576c` |
| 已通过、成功、完成 | `stat-blue` | 蓝色 `#4facfe → #00f2fe` |
| 已驳回、失败、警告 | `stat-orange` | 橙黄 `#fa709a → #fee140` |
| 进行中、活跃、启用 | `stat-green` | 绿色 `#43e97b → #38f9d7` |
| 其他状态 | `stat-cyan` | 青色 `#5ee7df → #b490ca` |

## 🔧 标准修改步骤

### 步骤1：替换HTML结构
1. 找到 `<el-row :gutter="20" class="stats-row">` 或类似结构
2. 将外层改为 `<div class="statistics-wrapper">`
3. 将每个 `<el-card shadow="hover" class="stat-card">` 改为 `<div class="stat-card stat-xxx">`
4. 移除所有内联 `style="background: ..."`
5. 将 `<div class="stat-info">` 改为 `<div class="stat-content">`
6. 调整 `:span` 值（3个卡片用8，4个卡片用6，2个卡片用12，1个卡片用24）

### 步骤2：添加样式引用
在文件末尾的 `<style lang="scss" scoped>` 中添加：
```scss
@import '@/assets/css/statistics-cards.scss';
```

### 步骤3：删除旧样式
删除页面中所有以下样式定义（如果存在）：
- `.stats-row`
- `.stat-card`（旧版）
- `.stat-content`（旧版）
- `.stat-icon`（旧版）
- `.stat-info`
- `.stat-label`（旧版）
- `.stat-value`（旧版）

**注意**：保留 `.tech-search-form`、`.tech-table`、`.tech-pagination` 等其他样式

## ⚠️ 特殊注意事项

### 1. 晋级关系页面（guanxi-list.vue）
这个页面使用的是**grid布局**（`.stats-cards`），需要转换为标准的**el-row栅格布局**

### 2. 卡片数量调整
- 3个卡片：`:span="8"`
- 4个卡片：`:span="6"`
- 2个卡片：`:span="12"`
- 1个卡片：`:span="24"`

### 3. 图标选择
使用 Element UI 官方图标，参考：https://element.eleme.cn/#/zh-CN/component/icon

常用图标：
- `el-icon-document` - 文档/记录
- `el-icon-time` - 时间/待审核
- `el-icon-circle-check` - 通过/成功
- `el-icon-circle-close` - 驳回/失败
- `el-icon-trophy` - 竞赛/奖杯
- `el-icon-user` - 用户/报名
- `el-icon-money` - 费用/金钱
- `el-icon-s-custom` - 团队/人员
