### 1. 数据库设计
- **表名**: `xiaoxi_tongzhi`
- **位置**: `Springboot_BYSJ/src/main/resources/db/xiaoxi_tongzhi.sql`
- **字段说明**:
  - `id`: 主键ID
  - `biaoti`: 消息标题
  - `neirong`: 消息内容
  - `leixing`: 消息类型（系统通知/审核结果/成绩通知/截止提醒）
  - `fasongren`: 发送人
  - `jieshouren_xuehao`: 接收人学号（学生）
  - `jieshouren_gonghao`: 接收人工号（教师）
  - `jieshouren_juese`: 接收人角色
  - `guanlian_id`: 关联业务ID
  - `guanlian_leixing`: 关联业务类型
  - `is_read`: 是否已读（已读/未读）
  - `read_time`: 阅读时间
  - `addtime`: 创建时间

### 2. 后端实现

#### 文件清单
| 文件 | 路径 | 说明 |
|------|------|------|
| Entity | `com/entity/XiaoxiTongzhiEntity.java` | 消息实体类 |
| View | `com/entity/view/XiaoxiTongzhiView.java` | 视图类 |
| Mapper | `com/dao/XiaoxiTongzhiDao.java` | 数据访问接口 |
| XML | `mapper/XiaoxiTongzhiDao.xml` | MyBatis映射文件 |
| Service | `com/service/XiaoxiTongzhiService.java` | 服务接口 |
| ServiceImpl | `com/service/impl/XiaoxiTongzhiServiceImpl.java` | 服务实现 |
| Controller | `com/controller/XiaoxiTongzhiController.java` | 控制器 |

#### API接口
| 接口 | 方法 | 说明 |
|------|------|------|
| `/xiaoxitongzhi/page` | GET | 分页查询消息列表 |
| `/xiaoxitongzhi/unread/count` | GET | 获取未读消息数量 |
| `/xiaoxitongzhi/markRead/{id}` | POST | 标记消息为已读 |
| `/xiaoxitongzhi/markAllRead` | POST | 批量标记所有消息为已读 |
| `/xiaoxitongzhi/delete` | POST | 删除消息（支持批量） |
| `/xiaoxitongzhi/send` | POST | 发送消息通知 |
