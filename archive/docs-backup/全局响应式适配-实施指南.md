## 🔧 如何应用到其他页面

### 方法一：手动添加（推荐用于少量页面）

#### 步骤：

1. **打开Vue文件**
   ```
   找到文件的 <style> 部分
   ```

2. **添加import语句**
   ```scss
   <style lang="scss" scoped>
   @import '@/assets/css/tech-theme.scss';
   @import '@/assets/css/global-responsive-mixin.scss';  // ← 添加这一行
   
   // 其他样式...
   </style>
   ```

3. **保存并测试**
   - 刷新浏览器
   - 使用开发者工具测试不同屏幕尺寸

#### 示例对比：

**修改前**:
```scss
<style lang="scss" scoped>
@import '@/assets/css/tech-theme.scss';

.action-wrapper {
  margin-bottom: 20px;
}
</style>
```

**修改后**:
```scss
<style lang="scss" scoped>
@import '@/assets/css/tech-theme.scss';
@import '@/assets/css/global-responsive-mixin.scss';  // ← 新增

.action-wrapper {
  margin-bottom: 20px;
}
</style>
```

### 方法二：批量添加（适用于大量页面）

#### Windows PowerShell脚本：

创建一个名为 `add-responsive.ps1` 的文件：

```powershell
# 设置路径
$basePath = "D:\BYSJ\BYSJ\Springboot_BYSJ\src\main\resources\admin\admin\src\views\modules"

# 查找所有list.vue文件
$files = Get-ChildItem -Path $basePath -Recurse -Filter "list.vue"

foreach ($file in $files) {
    $content = Get-Content $file.FullName -Raw -Encoding UTF8
    
    # 检查是否已包含
    if ($content -match "global-responsive-mixin") {
        continue
    }
    
    # 添加import
    $newContent = $content -replace '(<style lang="scss" scoped>)', "`$1`n@import '@/assets/css/global-responsive-mixin.scss';"
    
    if ($newContent -ne $content) {
        Set-Content -Path $file.FullName -Value $newContent -Encoding UTF8 -NoNewline
        Write-Host "Modified: $($file.Name)" -ForegroundColor Green
    }
}

Write-Host "Done!" -ForegroundColor Cyan
```

**执行命令**:
```powershell
cd D:\BYSJ\BYSJ\scripts
powershell -ExecutionPolicy Bypass -File add-responsive.ps1
```

## 🔍 常见问题

### Q1: 添加后样式没有变化？

**A**: 可能的原因：
1. 浏览器缓存 - 清除缓存或硬刷新 (Ctrl+F5)
2. SCSS未编译 - 重新运行 `npm run build`
3. 样式冲突 - 检查是否有更高优先级的样式

### Q2: 某些组件仍然不响应？

**A**: 可能需要针对特定组件添加额外样式：
```scss
::v-deep .el-specific-component {
  @media screen and (max-width: 768px) {
    // 自定义样式
  }
}
```

### Q3: 如何自定义断点？

**A**: 修改 `global-responsive-mixin.scss` 中的媒体查询：
```scss
@media screen and (max-width: 900px) {  // 自定义断点
  // 样式
}
```

### Q4: 会影响性能吗？

**A**: 不会。CSS媒体查询是原生支持的，性能影响微乎其微。
