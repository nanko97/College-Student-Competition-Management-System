# 团队人员变更功能使用说明

## 功能概述

团队人员变更功能允许学生提交团队成员变更申请（添加成员、移除成员、更换负责人），并由教师或管理员进行审核。审核通过后，系统会自动执行相应的变更操作。

## 业务流程

```
1. 学生提交变更申请 → 2. 填写变更信息 → 3. 教师/管理员审核 → 4. 审核通过自动执行变更
```

## 功能说明

### 一、学生端功能

#### 1. 人员变更申请列表
- **访问路径**：左侧菜单 → "人员变更管理" → "人员变更申请"
- **功能说明**：
  - 查看自己提交的所有变更申请
  - 显示申请状态（待审核/已通过/已驳回）
  - 支持按团队编号、变更类型、审核状态筛选
  - 可以撤销待审核的申请

#### 2. 提交变更申请
- **操作步骤**：
  1. 点击"申请变更"按钮
  2. 选择要变更的团队
  3. 选择变更类型：
     - **添加成员**：填写新成员的学号、姓名、学院、专业等信息
     - **移除成员**：从当前团队成员中选择要移除的成员
     - **更换负责人**：从当前团队成员中选择新负责人
  4. 填写变更原因（必填）
  5. 点击"提交申请"

#### 3. 查看申请详情
- **功能说明**：查看申请的详细信息，包括变更前后的数据对比

#### 4. 撤销申请
- **适用场景**：申请提交后但还未审核时，可以撤销申请
- **操作步骤**：在申请列表中点击"撤销"按钮

### 二、教师/管理员端功能

#### 1. 人员变更审核列表
- **访问路径**：左侧菜单 → "人员变更管理" → "人员变更审核"
- **功能说明**：
  - 查看所有待审核的变更申请
  - 支持按团队编号、变更类型筛选
  - 显示申请人信息、变更类型、申请时间等

#### 2. 审核变更申请
- **操作步骤**：
  1. 在审核列表中点击"通过"或"驳回"按钮
  2. 查看申请详情（团队信息、变更类型、变更原因等）
  3. 选择审核结果（通过/驳回）
  4. 填写审核意见（可选）
  5. 点击"确定"提交审核

#### 3. 查看变更历史
- **功能说明**：查看某个团队的所有变更历史记录

## 变更类型详解

### 1. 添加成员
**适用场景**：团队需要增加新成员

**操作流程**：
1. 学生选择"添加成员"类型
2. 填写新成员信息：
   - 学号（必填）
   - 姓名（必填）
   - 学院（必填）
   - 专业（必填）
   - 班级（可选）
3. 提交申请后等待审核
4. 审核通过后，系统自动：
   - 在团队成员表中添加新成员记录
   - 更新团队人数
   - 新成员角色默认为"队员"

**注意事项**：
- 新成员不能已经在其他活跃团队中
- 团队人数不能超过赛道设置的上限

### 2. 移除成员
**适用场景**：团队成员退出或被移除

**操作流程**：
1. 学生选择"移除成员"类型
2. 从当前团队成员列表中选择要移除的成员
3. 填写移除原因
4. 提交申请后等待审核
5. 审核通过后，系统自动：
   - 从团队成员表中删除该成员记录
   - 更新团队人数

**注意事项**：
- 负责人不能直接移除，需要先更换负责人
- 移除后该成员可以重新加入其他团队

### 3. 更换负责人
**适用场景**：团队需要更换队长/负责人

**操作流程**：
1. 学生选择"更换负责人"类型
2. 从当前团队成员列表中选择新负责人
3. 填写更换原因
4. 提交申请后等待审核
5. 审核通过后，系统自动：
   - 将原负责人角色改为"队员"
   - 将新成员角色改为"负责人"
   - 更新团队表中的负责人字段

**注意事项**：
- 只能从当前团队成员中选择新负责人
- 负责人不能直接退出团队，必须先转让负责人身份

## 常见问题

### Q1: 学生无法提交变更申请？
**A**: 请检查以下几点：
1. 是否选择了正确的团队
2. 变更类型是否选择
3. 变更原因是否填写
4. 如果是添加成员，新成员信息是否完整
5. 如果是移除成员或更换负责人，是否选择了正确的成员

### Q2: 教师看不到待审核的申请？
**A**: 审核列表只显示"待审核"状态的申请。如果申请已经被审核（通过或驳回），不会出现在审核列表中。可以在"人员变更申请"页面查看所有申请。

### Q3: 审核通过后团队成员没有变化？
**A**: 可能的原因：
1. 后端服务未正常启动
2. 数据库中缺少必要的表（jingsai_tuandui_chengyuan、jingsai_tuandui）
3. 查看后端日志，检查是否有错误信息

### Q4: 如何查看团队的变更历史？
**A**: 目前可以通过数据库直接查询：
```sql
SELECT * FROM jingsai_renyuan_biangueng 
WHERE tuandui_id = 团队ID 
ORDER BY addtime DESC;
```

### Q5: 负责人可以移除自己吗？
**A**: 不可以。负责人如果要退出团队，必须先更换负责人，然后由新负责人将其移除，或者负责人申请更换负责人后以普通队员身份退出。

## 与现有功能的关联

### 1. 团队管理模块
- 人员变更依赖团队记录（`jingsai_tuandui` 表）
- 变更后会影响团队成员列表（`jingsai_tuandui_chengyuan` 表）

### 2. 竞赛报名模块
- 团队通常与竞赛报名关联
- 人员变更可能影响报名信息的准确性

### 3. 晋级管理模块
- 团队晋级后可能需要调整人员
- 变更记录可以帮助追踪团队演变过程
