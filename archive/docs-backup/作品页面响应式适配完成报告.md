## 🎯 响应式设计原则

### 1. 移动优先 (Mobile First)
- 基础样式针对小屏幕设计
- 通过媒体查询逐步增强大屏幕体验

### 2. 弹性布局 (Flexible Layout)
- 使用百分比宽度而非固定像素
- 使用Flexbox和Grid布局
- 内容自适应容器宽度

### 3. 渐进增强 (Progressive Enhancement)
- 核心功能在所有设备上可用
- 大屏幕提供额外的视觉增强
- 小屏幕优化触摸体验

### 4. 内容优先 (Content First)
- 重要内容始终可见
- 次要内容在小屏幕上可隐藏
- 保持内容的可读性和可操作性

## 🔧 技术实现细节

### 1. CSS媒体查询断点

```css
/* 平板设备 */
@media screen and (max-width: 1200px) { }

/* 手机设备 */
@media screen and (max-width: 768px) { }

/* 超小屏幕 */
@media screen and (max-width: 480px) { }
```

### 2. 关键CSS属性

- `box-sizing: border-box` - 统一盒模型
- `overflow-x: auto` - 横向滚动支持
- `-webkit-overflow-scrolling: touch` - 平滑滚动
- `flex-wrap: wrap` - 弹性换行
- `word-break: break-word` - 长文本换行

### 3. Vue深度选择器

使用 `::v-deep` 修改Element UI组件内部样式：

```scss
::v-deep .el-dialog {
  width: 95% !important;
}
```
