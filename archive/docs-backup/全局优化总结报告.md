```java

    // 多表操作、状态变更、审核操作都需要事务

```

### 2. 常量提取
```java
// 避免硬编码
final String STATUS_APPROVED = "已通过";

```

### 3. 防重复机制
```java
// 生成唯一记录前检查是否存在
Integer count = service.selectCount(wrapper);
if (count > 0) {
    log.warn("记录已存在，跳过生成");
    return;
}
```

### 4. 异常分类
```java
} catch (RuntimeException e) {
    throw e; // 业务异常，触发回滚
} catch (Exception e) {
    throw new RuntimeException("操作失败", e); // 系统异常
}
```

### 5. 日志规范
```java
log.info("✓ 操作成功，ID: {}", id);  // 成功用✓
log.error("✗ 操作失败", e);          // 失败用✗
```
