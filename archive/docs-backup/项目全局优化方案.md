### 二、Service层优化

#### 2.1 提取公共方法
**建议创建 BaseService**：
```java
public abstract class BaseService<T> {
    
    /**
     * 通用分页查询
     */
    public PageUtils queryPage(Map<String, Object> params, Wrapper<T> wrapper) {
        // 统一分页逻辑
    }
    
    /**
     * 通用数据权限过滤
     */
    protected void applyDataPermission(Wrapper<T> wrapper, String tableName, String username) {
        if ("jiaoshi".equals(tableName)) {
            wrapper.eq("gonghao", username);
        } else if ("xuesheng".equals(tableName)) {
            wrapper.eq("xuehao", username);
        }
    }
}
```

#### 2.2 优化查询性能
**问题**：N+1查询问题
**解决方案**：
- 使用关联查询代替多次单表查询
- 添加必要的索引
- 使用缓存（Redis）

**示例**：
```java
// 优化前：N+1查询
List<JingsaibaomingEntity> list = baomingService.selectList(wrapper);
for (JingsaibaomingEntity item : list) {
    JingsaixinxiEntity jingsai = jingsaiService.selectById(item.getJingsaiId());
    item.setJingsai(jingsai);
}

// 优化后：一次关联查询
List<JingsaibaomingVO> list = baomingDao.selectWithJingsai(wrapper);
```

### 七、前端优化（Vue）

#### 7.1 统一API调用
**建议创建**：`src/api/index.js`
```javascript
import axios from 'axios'

const request = axios.create({
  baseURL: process.env.VUE_APP_BASE_API,
  timeout: 30000
})

// 请求拦截器
request.interceptors.request.use(config => {
  const token = localStorage.getItem('token')
  if (token) {
    config.headers['Authorization'] = token
  }
  return config
})

// 响应拦截器
request.interceptors.response.use(response => {
  const res = response.data
  if (res.code !== 0) {
    Message.error(res.msg || '请求失败')
    return Promise.reject(new Error(res.msg || '请求失败'))
  }
  return res
})

export default request
```

### 九、日志优化

#### 9.1 统一日志格式
**当前**：部分Controller有日志，部分没有
**优化**：所有Controller的关键操作都记录日志

**日志级别规范**：
- DEBUG：调试信息、查询结果
- INFO：关键业务流程、操作成功
- WARN：警告信息、非预期但可恢复
- ERROR：错误信息、异常堆栈

#### 9.2 添加性能监控
**建议**：
```java
@Around("@annotation(com.annotation.PerformanceMonitor)")
public Object monitorPerformance(ProceedingJoinPoint joinPoint) throws Throwable {
    long start = System.currentTimeMillis();
    Object result = joinPoint.proceed();
    long duration = System.currentTimeMillis() - start;
    
    if (duration > 1000) {
        log.warn("方法执行缓慢: {}, 耗时: {}ms", 
                joinPoint.getSignature(), duration);
    }
    
    return result;
}
```
