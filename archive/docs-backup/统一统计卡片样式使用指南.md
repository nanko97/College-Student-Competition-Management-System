# 统一统计卡片样式使用指南

## 📋 概述

所有页面的数据展板已统一使用**现代化渐变卡片样式**，设计灵感来自"我的复核记录"页面。

---

## 🎨 样式特点

- ✅ 现代化渐变背景（6种配色方案）
- ✅ 悬停动画效果（上浮+阴影）
- ✅ 响应式设计（平板/手机自适应）
- ✅ 图标+数字+标签的经典布局
- ✅ 白色文字+阴影增强可读性

---

## 🎯 配色方案

| 类名 | 渐变色 | 用途 |
|------|--------|------|
| `stat-total` / `stat-purple` | 紫色渐变 `#667eea → #764ba2` | 总数、总计 |
| `stat-pending` / `stat-pink` | 粉红渐变 `#f093fb → #f5576c` | 待审核、待处理 |
| `stat-approved` / `stat-blue` | 蓝色渐变 `#4facfe → #00f2fe` | 已通过、成功 |
| `stat-rejected` / `stat-orange` | 橙黄渐变 `#fa709a → #fee140` | 已驳回、警告 |
| `stat-active` / `stat-green` | 绿色渐变 `#43e97b → #38f9d7` | 进行中、活跃 |
| `stat-cyan` | 青色渐变 `#5ee7df → #b490ca` | 其他状态 |

---

## 📝 使用方法

### 1. 引入样式文件

在页面 `<style>` 标签中添加：

```scss
@import '@/assets/css/statistics-cards.scss';
```

### 2. HTML结构

```html
<!-- 数据统计 -->
<div class="statistics-wrapper">
  <el-row :gutter="20">
    <el-col :span="6">
      <div class="stat-card stat-total">
        <div class="stat-icon"><i class="el-icon-document"></i></div>
        <div class="stat-content">
          <div class="stat-value">{{ totalCount }}</div>
          <div class="stat-label">总申请数</div>
        </div>
      </div>
    </el-col>
    <el-col :span="6">
      <div class="stat-card stat-pending">
        <div class="stat-icon"><i class="el-icon-time"></i></div>
        <div class="stat-content">
          <div class="stat-value">{{ pendingCount }}</div>
          <div class="stat-label">待审核</div>
        </div>
      </div>
    </el-col>
    <el-col :span="6">
      <div class="stat-card stat-approved">
        <div class="stat-icon"><i class="el-icon-circle-check"></i></div>
        <div class="stat-content">
          <div class="stat-value">{{ approvedCount }}</div>
          <div class="stat-label">已通过</div>
        </div>
      </div>
    </el-col>
    <el-col :span="6">
      <div class="stat-card stat-rejected">
        <div class="stat-icon"><i class="el-icon-circle-close"></i></div>
        <div class="stat-content">
          <div class="stat-value">{{ rejectedCount }}</div>
          <div class="stat-label">已驳回</div>
        </div>
      </div>
    </el-col>
  </el-row>
</div>
```

### 3. 删除旧样式

删除页面中旧的统计卡片样式代码（通常是 `.stats-row`、`.stat-card` 等自定义样式）

## 🔄 批量替换步骤

### 步骤1：替换HTML结构

**旧结构**（需要删除）：
```html
<el-row :gutter="20" class="stats-row">
  <el-col :span="8">
    <el-card shadow="hover" class="stat-card">
      <div class="stat-content">
        <div class="stat-icon" style="background: ...">
          <i class="el-icon-document"></i>
        </div>
        <div class="stat-info">
          <div class="stat-label">总数</div>
          <div class="stat-value">{{ total }}</div>
        </div>
      </div>
    </el-card>
  </el-col>
</el-row>
```

**新结构**（使用全局样式）：
```html
<div class="statistics-wrapper">
  <el-row :gutter="20">
    <el-col :span="6">
      <div class="stat-card stat-total">
        <div class="stat-icon"><i class="el-icon-document"></i></div>
        <div class="stat-content">
          <div class="stat-value">{{ total }}</div>
          <div class="stat-label">总数</div>
        </div>
      </div>
    </el-col>
  </el-row>
</div>
```

### 步骤2：引入样式文件

在 `<style lang="scss" scoped>` 中添加：

```scss
@import '@/assets/css/statistics-cards.scss';
```

### 步骤3：删除旧样式

删除页面中所有与统计卡片相关的样式代码，例如：
- `.stats-row`
- `.stat-card`
- `.stat-content`
- `.stat-icon`
- `.stat-info`
- `.stat-label`
- `.stat-value`

---

## ⚠️ 注意事项

1. **栅格布局调整**：新样式默认4列（`:span="6"`），旧样式可能是3列（`:span="8"`），请根据实际卡片数量调整
2. **图标选择**：使用 Element UI 图标类名，如 `el-icon-document`、`el-icon-time` 等
3. **数据绑定**：确保 `stat-value` 中的数据变量已正确绑定
4. **响应式**：样式已内置响应式断点（1200px、768px），无需额外处理
