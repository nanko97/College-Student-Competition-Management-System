## 四、接口调用方式验证

所有前端API调用均使用统一的 Alova HTTP 客户端：

```typescript
import { Alova } from '@/utils/http/alova'

// GET 请求示例
export function getList(params: any) {
  return Alova.Get('/xxx/page', { 
    params, 
    meta: { isTransformResponse: false } 
  })
}

// POST 请求示例
export function saveData(data: any) {
  return Alova.Post('/xxx/save', data, { 
    meta: { isTransformResponse: false } 
  })
}
```

**响应格式统一**:
```json
{
  "code": 0,
  "msg": "success",
  "page": {
    "list": [],
    "total": 100,
    "currPage": 1,
    "pageSize": 10
  }
}
```
