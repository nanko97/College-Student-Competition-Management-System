# 作品管理模块使用说明

## 功能概述

为完善竞赛管理系统的业务流程，新增了**作品管理模块**，实现学生提交参赛作品、教师查看和评分的完整流程。

## 业务流程

```
1. 学生报名竞赛 → 2. 教师审核报名 → 3. 学生提交作品 → 4. 教师查看/下载作品 → 5. 教师评分 → 6. 学生查看成绩
```

## 功能说明

### 一、学生端功能

#### 1. 我的作品列表
- **访问路径**：左侧菜单 → "我的作品"
- **功能说明**：
  - 查看所有已报名且审核通过的竞赛
  - 显示每个竞赛的作品提交状态（已提交/未提交）
  - 支持按竞赛名称筛选

#### 2. 提交作品
- **操作步骤**：
  1. 在"我的作品"列表中找到需要提交作品的竞赛
  2. 点击"提交作品"按钮
  3. 选择作品文件（支持doc、docx、pdf、zip、rar、7z、ppt、pptx、xls、xlsx格式）
  4. 文件大小不超过50MB
  5. 点击"确定上传"

#### 3. 更新作品
- **适用场景**：作品截止日期前，可以重新上传作品覆盖之前的版本
- **操作步骤**：
  1. 在"我的作品"列表中找到已提交作品的竞赛
  2. 点击"更新作品"按钮
  3. 选择新的作品文件
  4. 点击"确定上传"（会自动覆盖旧文件）

#### 4. 下载作品
- **功能说明**：学生可以下载自己已提交的作品文件进行查看

### 二、教师端功能

#### 1. 作品管理
- **访问路径**：左侧菜单 → "作品管理"
- **功能说明**：
  - 查看所有学生提交的参赛作品
  - 支持按竞赛名称、学生姓名筛选
  - 显示作品提交统计信息（已提交数、未提交数）

#### 2. 下载作品
- **操作步骤**：
  1. 在作品管理列表中找到需要查看的作品
  2. 点击"下载作品"按钮
  3. 浏览器会自动下载该学生的作品文件

#### 3. 作品评分
- **操作步骤**：
  1. 在作品管理列表中找到需要评分的学生
  2. 点击"评分"按钮
  3. 输入作品评分（0-100分）
  4. 填写评价内容（可选）
  5. 点击"确定评分"

- **注意事项**：
  - 如果学生已有评分，系统会提示是否重新评分
  - 重新评分会覆盖之前的分数

#### 4. 查看评分
- **功能说明**：查看某个学生的历史评分记录
- **操作步骤**：
  1. 在作品管理列表中点击"查看评分"按钮
  2. 弹窗显示评分详情（分数、评价内容、评分教师、评分时间等）

## 技术实现

### 后端接口

| 接口路径 | 请求方式 | 功能说明 | 权限 |
|---------|---------|---------|------|
| `/zuopin/my/list` | GET | 查询我的作品列表（学生端） | 学生 |
| `/zuopin/list` | GET | 查询所有作品列表（教师端） | 教师/管理员 |
| `/zuopin/upload` | POST | 上传/更新作品 | 学生 |
| `/zuopin/delete` | POST | 删除已提交的作品 | 学生 |
| `/zuopin/statistics` | GET | 统计作品提交情况 | 学生/教师 |

### 文件存储

- **存储路径**：项目根目录 `/upload/zuopin/`
- **文件命名规则**：`学号_报名ID_时间戳.扩展名`
  - 例如：`2021001_1234567890_1714867200000.pdf`
- **数据库字段**：`jingsaibaoming.cansaizuopin`
  - 存储相对路径：`upload/zuopin/文件名`

### 前端页面

- **教师端**：`src/views/modules/zuopin/list.vue`

### 路由配置

- 学生端路由：`/xuesheng-my-zuopin`
- 教师端路由：`/zuopin`

### 菜单配置

已在 `menu.js` 中添加：
- 学生菜单："我的作品"

## 常见问题

### Q1: 学生无法上传作品？
**A**: 请检查以下几点：
1. 报名信息是否已通过教师审核（`sfsh = '通过'`）
2. 文件格式是否符合要求（doc、docx、pdf、zip、rar、7z、ppt、pptx、xls、xlsx）
3. 文件大小是否超过50MB限制
4. 是否为本人报名记录（只能操作自己的报名）

### Q2: 教师看不到学生提交的作品？
**A**: 作品管理列表只显示：
1. 报名审核通过的记录
2. 已提交作品的记录（`cansaizuopin` 字段不为空）

### Q3: 如何修改已提交的作品？
**A**: 学生可以在"我的作品"列表中点击"更新作品"按钮，重新上传文件即可覆盖旧版本。

### Q4: 评分后可以修改吗？
**A**: 教师可以重新评分，系统会提示确认是否覆盖原有分数。

## 数据流转示例

### 学生提交流程
```
1. 学生登录 → 进入"我的作品"页面
2. 查看已报名的竞赛列表
3. 点击"提交作品" → 选择文件 → 上传
4. 后端保存文件到 /upload/zuopin/ 目录
5. 更新 jingsaibaoming 表的 cansaizuopin 字段
6. 前端刷新列表，显示"已提交"状态
```

### 教师评分流程
```
1. 教师登录 → 进入"作品管理"页面
2. 查看学生提交的作品列表
3. 点击"下载作品" → 查看学生作品
4. 点击"评分" → 输入分数和评价 → 提交
5. 后端在 zuopindafen 表中创建评分记录
6. 学生可在"我的成绩"中查看评分结果
```

## 与现有功能的关联

### 1. 竞赛报名模块
- 作品管理依赖报名记录（`jingsaibaoming` 表）
- 只有审核通过的报名才能提交作品

### 2. 作品打分模块
- 教师在作品管理中可以直接对学生评分
- 评分数据存储到 `zuopindafen` 表
- 学生可在"我的成绩"中查看

### 3. 成绩复核模块
- 学生对评分有异议可申请复核
- 复核申请关联到具体的评分记录
