# 菜单命名规范化方案

## 📋 优化时间
2026-05-18

## 🎯 优化目标
统一三个角色（学生、教师、管理员）的菜单命名规范，使菜单名称更符合各角色的业务视角。

---

## 📊 命名规范

### 1️ 学生角色（我的XX）
**命名原则**：强调个人视角，使用"我的XX"（例外：竞赛信息是公共资源）

| 一级菜单 | 二级菜单 | 说明 |
|---------|---------|------|
| 竞赛信息 | 竞赛信息 | 浏览所有可报名的竞赛（公共资源） |
| 我的报名 | 我的报名 | 查看个人报名记录 |
| 我的缴费 | 我的缴费 | 查看个人缴费记录 |
| 我的作品 | 我的作品 | 提交和查看个人作品 |
| 我的晋级 | 我的晋级 | 查看个人晋级记录 |
| 我的成绩 | 我的成绩 | 查看个人竞赛成绩 |
| 我的团队 | 我的团队 | 查看和管理个人团队 |
|          | 申请加入团队 | 申请加入其他团队 |
|          | 创建团队 | 创建新团队 |
| 我的复核 | 我的复核 | 查看个人成绩复核申请 |

### 2️⃣ 教师角色（XX管理）
**命名原则**：强调管理视角，使用"XX管理"

| 一级菜单 | 二级菜单 | 说明 |
|---------|---------|------|
| 竞赛管理 | 竞赛信息 | 发布和管理竞赛 |
| 报名管理 | 竞赛报名 | 审核学生报名 |
| 作品管理 | 作品打分 | 给学生作品打分 |
|          | 成绩复核审核 | 审核学生的复核申请 |
|          | 作品查看 | 查看和下载学生作品 |
| 费用管理 | 缴费记录 | 查看学生缴费记录 |
|          | 缴费审核 | 审核学生缴费凭证 |
| 晋级管理 | 晋级关系 | 查看晋级关系配置 |
|          | 晋级审核 | 审核学生晋级申请 |
| 级别认定管理 | 级别版本 | 查看竞赛级别版本 |
| 赛道管理 | 赛道管理 | 管理竞赛赛道 |
| 团队管理 | 团队管理 | 查看团队信息 |
| 人员变更管理 | 人员变更申请 | 查看人员变更申请 |
|              | 人员变更审核 | 审核人员变更 |

### 3️⃣ 管理员角色（XX管理 + 用户管理）
**命名原则**：全局管理视角，包含用户管理功能

| 一级菜单 | 二级菜单 | 说明 |
|---------|---------|------|
| 用户管理 | 学生 | 管理学生账号 |
|          | 教师 | 管理教师账号 |
| 竞赛管理 | 竞赛信息 | 全局竞赛管理 |
| 报名管理 | 竞赛报名 | 全局报名管理 |
| 费用管理 | 费用配置 | 配置竞赛费用 |
|          | 缴费记录 | 查看所有缴费记录 |
|          | 缴费审核 | 审核缴费凭证 |
| 晋级管理 | 晋级关系 | 配置晋级关系 |
|          | 晋级审核 | 审核晋级申请 |
| 级别认定管理 | 级别版本 | 管理级别版本 |
| 赛道管理 | 赛道管理 | 管理竞赛赛道 |
| 团队管理 | 团队管理 | 查看所有团队 |
| 作品管理 | 作品打分 | 管理作品打分 |
|          | 作品查看 | 查看和下载作品 |
| 复核管理 | 成绩复核 | 管理成绩复核 |
| 人员变更管理 | 人员变更申请 | 查看变更申请 |
|              | 人员变更审核 | 审核变更申请 |

## ✨ 主要优化点

### 优化1：学生菜单统一"我的"前缀
**优化前**：
- ❌ 竞赛信息管理
- ❌ 竞赛报名管理
- ❌ 团队管理

**优化后**：
- ✅ 竞赛信息（公共资源，不使用“我的”）
- ✅ 我的报名
- ✅ 我的团队

### 优化2：教师/管理员菜单精简
**优化前**：
- ❌ 竞赛信息管理（太长）
- ❌ 竞赛报名管理（太长）
- ❌ 赛道分组管理（冗余）
- ❌ 成绩复核管理（与教师视角不符）

**优化后**：
- ✅ 竞赛管理
- ✅ 报名管理
- ✅ 赛道管理
- ✅ 复核管理（管理员）

### 优化3：管理员新增"用户管理"
**优化前**：
- ❌ 学生管理（单独一级菜单）
- ❌ 教师管理（单独一级菜单）

**优化后**：
- ✅ 用户管理（统一管理学生和教师）
  - 学生
  - 教师

### 优化4：作品管理整合
**优化前（教师）**：
- ❌ 作品打分管理（一级菜单）
- ❌ 作品管理（一级菜单）

**优化后**：
- ✅ 作品管理（一级菜单）
  - 作品打分
  - 成绩复核审核
  - 作品查看
