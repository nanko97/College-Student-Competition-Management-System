### 1. 动态菜单生成系统

#### 📋 菜单生成工具 (`src/utils/menuGenerator.ts`) - 474行

创建了完整的动态菜单管理系统：

##### 核心功能

| 函数 | 说明 | 使用场景 |
|------|------|---------|
| `filterMenusByPermission()` | 根据权限过滤菜单 | 登录后生成个性化菜单 |
| `generateMenusFromRoutes()` | 从路由配置生成菜单 | 自动同步路由和菜单 |
| `convertToNaiveMenu()` | 转换为Naive UI格式 | 渲染侧边栏菜单 |
| `generateBreadcrumbs()` | 生成面包屑导航 | 页面顶部导航 |
| `getDefaultMenusByRole()` | 获取角色默认菜单 | 初始化菜单 |

##### 角色菜单配置

**管理员菜单** (ADMIN_DEFAULT_MENUS)：
- ✅ 仪表盘
- ✅ 竞赛管理（信息/报名/缴费/晋级/赛道/级别）
- ✅ 团队管理（列表/人员变更）
- ✅ 作品管理（列表/打分/复核）
- ✅ 用户管理（学生/教师）

**教师菜单** (TEACHER_DEFAULT_MENUS)：
- ✅ 仪表盘
- ✅ 竞赛管理（信息/报名审核/缴费审核/晋级）
- ✅ 团队管理（列表/人员变更审核）
- ✅ 作品管理（列表/打分/复核）
- ❌ 用户管理（无权限）

**学生菜单** (STUDENT_DEFAULT_MENUS)：
- ✅ 仪表盘
- ✅ 竞赛信息（查看）
- ✅ 我的报名
- ✅ 我的缴费
- ✅ 我的晋级
- ✅ 我的作品
- ✅ 我的团队

**使用示例**：

```vue
<script setup lang="ts">
import { 
  getDefaultMenusByRole, 
  filterMenusByPermission,
  convertToNaiveMenu 
} from '@/utils/menuGenerator'
import { useUser } from '@/store/modules/user'

const userStore = useUser()

// 获取当前角色的默认菜单
const defaultMenus = getDefaultMenusByRole(userStore.getRole)

// 根据权限过滤菜单
const filteredMenus = filterMenusByPermission(defaultMenus)

// 转换为Naive UI格式
const menuOptions = convertToNaiveMenu(filteredMenus)
</script>

<template>
  <NMenu :options="menuOptions" />
</template>
```

##### 面包屑导航

```vue
<script setup lang="ts">
import { generateBreadcrumbs } from '@/utils/menuGenerator'
import { useRoute } from 'vue-router'

const route = useRoute()
const menus = ref([]) // 从后端或配置获取

const breadcrumbs = computed(() => 
  generateBreadcrumbs(route.path, menus.value)
)
</script>

<template>
  <NBreadcrumb>
    <NBreadcrumbItem 
      v-for="item in breadcrumbs" 
      :key="item.path"
    >
      {{ item.title }}
    </NBreadcrumbItem>
  </NBreadcrumb>
</template>
```

**收益**：
- ✅ 自动化菜单管理
- ✅ 基于角色的菜单隔离
- ✅ 权限细粒度控制
- ✅ 面包屑自动生成

### 2. 富文本编辑器组件

#### 📝 富文本编辑器 (`src/components/RichTextEditor/index.vue`) - 290行

基于 `@vueup/vue-quill` 封装的富文本编辑器：

##### 功能特性

- 📄 完整的富文本编辑功能
- 🎨 支持标题/粗体/斜体/下划线/删除线
- 🌈 文字颜色和背景色
- 📋 有序/无序列表
- 📐 对齐方式
- 🔗 链接/图片/视频插入
- 📊 字数统计
- ⚙️ 可配置工具栏
- 📱 移动端优化
- ♿ 只读模式支持

##### Props配置

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| modelValue | string | '' | 编辑器内容 |
| theme | 'snow' \| 'bubble' | 'snow' | 主题 |
| placeholder | string | '请输入内容...' | 占位符 |
| maxLength | number | undefined | 最大长度 |
| showCount | boolean | true | 显示字数统计 |
| readonly | boolean | false | 只读模式 |
| height | string | '300px' | 编辑器高度 |
| toolbar | any[] | 默认工具栏 | 自定义工具栏 |

##### 暴露方法

| 方法 | 说明 | 返回值 |
|------|------|--------|
| `getEditor()` | 获取Quill实例 | Quill Editor |
| `insertText(index, text)` | 插入文本 | void |
| `insertImage(url)` | 插入图片 | void |
| `clearContent()` | 清空内容 | void |

| `getText()` | 获取纯文本 | string |

### 3. 按钮级权限控制系统

#### 🔒 权限指令 (`src/directives/permissionButton.ts`) - 166行

实现了细粒度的按钮级权限控制：

##### 权限指令列表

| 指令 | 说明 | 使用场景 |
|------|------|---------|
| `v-permission-btn:visible` | 无权限时隐藏 | 操作按钮 |
| `v-permission-btn:disabled` | 无权限时禁用 | 敏感操作 |
| `v-audit` | 审核权限（教师+管理员） | 审核按钮 |
| `v-score` | 打分权限（教师+管理员） | 打分按钮 |
| `v-manage-users` | 用户管理权限（仅管理员） | 用户管理按钮 |

##### 使用示例

**基础权限控制**：

```vue
<template>
  <!-- 无权限时隐藏 -->
  <NButton 
    v-permission-btn:visible="'competition:info:add'"
    @click="handleAdd"
  >
    新增竞赛
  </NButton>
  
  <!-- 无权限时禁用 -->
  <NButton 
    v-permission-btn:disabled="'competition:info:edit'"
    @click="handleEdit"
  >
    编辑竞赛
  </NButton>
  
  <!-- 支持权限数组 -->
  <NButton 
    v-permission-btn:visible="['competition:info:edit', 'competition:info:delete']"
    @click="handleDelete"
  >
    删除竞赛
  </NButton>
</template>
```

**角色权限控制**：

```vue
<template>
  <!-- 仅教师和管理员可见 -->
  <NButton v-audit @click="handleAudit">
    审核报名
  </NButton>
  
  <!-- 仅教师和管理员可见 -->
  <NButton v-score @click="handleScore">
    作品打分
  </NButton>
  
  <!-- 仅管理员可见 -->
  <NButton v-manage-users @click="handleManageUsers">
    用户管理
  </NButton>
</template>
```

**在表格中使用**：

```vue
<template>
  <NDataTable :columns="columns" :data="tableData">
    <template #action="{ row }">
      <NSpace>
        <NButton 
          v-permission-btn:visible="'competition:registration:audit'"
          size="small"
          @click="handleAudit(row)"
        >
          审核
        </NButton>
        
        <NButton 
          v-permission-btn:visible="'competition:registration:reject'"
          size="small"
          type="error"
          @click="handleReject(row)"
        >
          拒绝
        </NButton>
      </NSpace>
    </template>
  </NDataTable>
</template>
```

**收益**：
- ✅ 细粒度权限控制
- ✅ 防止越权操作
- ✅ 提升系统安全性
- ✅ 用户体验优化（隐藏无关按钮）
