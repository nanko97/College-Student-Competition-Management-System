## 🚀 测试准备

### 1. 启动服务

#### 启动后端
```bash
cd Springboot_BYSJ
mvn spring-boot:run
```
确认后端启动成功，访问：http://localhost:9090/BYSJ_Springboot

#### 启动前端（新窗口）
```bash
cd new-admin
pnpm run dev
```
确认前端启动成功，访问：http://localhost:8081
